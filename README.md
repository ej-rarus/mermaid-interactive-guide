# 🎨 Mermaid 다이어그램 - 인터랙티브 가이드

텍스트로 다이어그램을 그리는 **Mermaid**를 5분만에 배워보세요!

## ✨ 특징

- 🎯 **초보자 친화적**: 코딩 경험 없어도 OK!
- 🎨 **인터랙티브**: 직접 코드를 수정하고 결과를 바로 확인
- 🚀 **Claude 연동**: AI에게 시키면 자동으로 다이어그램 생성
- 💼 **실무 활용**: 기획자, 개발자, PM을 위한 실전 예시
- 📱 **반응형**: 모바일, 태블릿, 데스크톱 모두 지원

## 🚀 바로 시작하기

👉 **[가이드 시작하기](https://mermaid-interactive-guide.vercel.app)**

또는 로컬에서 실행:

```bash
# 레포 클론
git clone https://github.com/yourusername/mermaid-guide.git

# 디렉토리 이동
cd mermaid-guide

# index.html 파일을 브라우저로 열기
open index.html  # Mac
# 또는
start index.html # Windows
```

## 🛠️ 기술 스택

- **Mermaid.js** v10 - 다이어그램 렌더링
- **Vanilla JavaScript** - 인터랙티브 기능
- **CSS3** - 반응형 디자인

## 📦 배포하기

### Vercel (추천)
1. GitHub에 레포 push
2. [Vercel](https://vercel.com)에서 "New Project" 클릭
3. GitHub 레포 선택
4. Deploy!

### Netlify
1. GitHub에 레포 push
2. [Netlify](https://netlify.com)에서 "Add new site" 클릭
3. GitHub 레포 선택
4. Deploy!

### GitHub Pages
1. Settings → Pages
2. Source를 "main" 브랜치로 설정
3. Save!
